# FitControl Pro 2.0

Projeto recriado do zero com arquitetura limpa: React + Vite no frontend e Node.js + Express no backend.

## Rodar localmente

### Backend
```bash
cd backend
copy .env.example .env
npm install
npm run dev
```

### Frontend
```bash
cd frontend
copy .env.example .env
npm install
npm run dev
```

Frontend: http://localhost:5173  
Backend: http://localhost:4000

## Usuário inicial
Crie um usuário em `/register`.

## Módulos
- Autenticação JWT
- Dashboard
- Alunos
- Avaliações físicas
- Motor de cálculos
- Treinos
- IA inicial por regras para geração de treinos
- Evolução
- Relatórios
- Backup/exportação/importação JSON
